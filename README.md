# AI-Based Face Recognition Attendance System using MATLAB

## Overview

This project is a Face Recognition Attendance System developed using MATLAB, Computer Vision Toolbox, and Machine Learning techniques.

The system detects faces, recognizes individuals from a custom dataset, and automatically records attendance in a CSV file.

## Features

* Face Detection
* Face Recognition using KNN Classifier
* Multi-user Dataset Support
* Automatic Attendance Recording
* MATLAB-based Implementation
* 85.71% Recognition Accuracy

## Technologies Used

* MATLAB
* Computer Vision Toolbox
* Image Processing Toolbox
* Statistics and Machine Learning Toolbox
